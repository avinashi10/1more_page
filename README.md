## Getting Started
1. Clone the repository
```
git clone https://github.com/avinashi10/1more_page.git
```
2. Install all the dependencies
```
npm install
```
3. Start Webpack and the server
```
npm run client-dev
npm run server-dev
```
4. Navigate to localhost:3000 in your local browser
